Hello, I'm Stathis Kaloutsidis and I developed this app for education purposes and as Proof of Concept (POC).
This web application collects users height via a HTML form and send the data to a secure PostgreSQL database. Then it send an email back to the user with statistical data about the average height of all records that are stored in the db.
Tech Stack: Python, Flask & PostgreSQL.
